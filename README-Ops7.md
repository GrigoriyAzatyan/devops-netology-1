#### 3.
Ethernet Frame = 1518 byte
MTU = 1518 - 14(Ethernet Header) + 4 (Ethernet Trailer) = 1500 byte 
payload = 1500 - (20(IP Header) + 20(TCP Header)) = 1460 bytes

(1460 / 1518) * 100 = 96.18% (96,18 Mbit/s)
